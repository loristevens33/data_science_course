For this project, I am creating a movie recommendation system using the MovieLens dataset. The objective is to train a machine learning algorithm to predict moving ratings. 

The dataset used is a subset of the 10M version of the MovieLens dataset. The full MovieLens dataset is available at the following location:

https://grouplens.org/datasets/movielens/latest/

The following documents are available:

code.R - full code for data set creation, analysis and modeling

report.Rmd - R Markdown report including code and detailed insights
